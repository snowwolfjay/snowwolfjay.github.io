## HCZ Material theme

基于一个jekyll Material 模板的个人博客站点，测试用。

### Demo
* [https://codeasashu.github.io/hcz-jekyll-blog](https://codeasashu.github.io/hcz-jekyll-blog/)

#### Feature

* Sitemap and XML Feed
* Site Search 
* Projects and tags
* Paginations in homepage
* Posts under category
* Related Posts
* Highlight pre
* Next & Previous Post
* Disqus comment
